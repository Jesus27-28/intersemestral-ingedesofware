¿QUE ES  UN Archivo README?
un archivo readme es un archivo de modo texto esto puede contener información de un software como instrucciones de instalación,configuracion otra información para los usuarios o desarrolladores que trabajen en esto.
en pocas palabras es una guía que los desarrolladores incluyan de como utilizar su software de manera correcta.



¿QUE ES GIT IGNORE?
 se utiliza para repositorios  de git esto para que no se agregue archivos que se deben ignorar solo los archivos específicos sobre el software
